Refresh Jobs Data

How to Update Jobs

The job listings are scraped from various websites. To refresh the data, you need to run the scraper script.

Option 1: Run All Scrapers (Default)

cd C:\Python\desktop_python_files\other_script\mobile
python run_scrapers.py

Option 2: Run Specific Scrapers

python run_scrapers.py --spiders onlinejobs remoteok freelancer

Option 3: Run All Available Scrapers

python run_scrapers.py --all

Option 4: Limit Pages to Scrape

python run_scrapers.py --pages 5

Available Scrapers

  • onlinejobs - OnlineJobs.ph (Philippines remote jobs)
  • remoteok - RemoteOK (Remote tech jobs)
  • weworkremotely - We Work Remotely (Remote jobs)
  • freelancer - Freelancer.com (Freelance projects)
  • guru - Guru.com (Freelance jobs)
  • peopleperhour - PeoplePerHour (Freelance projects)
  • indeed - Indeed (Various jobs)
  • hubstaff - Hubstaff Talent (Remote jobs)

After Running the Scraper

Once the scraper completes:

  1. The jobs will be saved to scraped_jobs.json in the parent directory
  2. Run update_jobs.bat to copy the data into the jobs_flask folder
  3. Refresh this website to see the new jobs
  4. No need to restart the Flask server

Quick Update Command

Windows - Double click this file:

update_jobs.bat

Or manually copy scraped_jobs.json from the parent directory into the jobs_flask folder.
